The Class-ic Detective

Imagine a murder mystery game in which the player gets to choose a detective. To create the detective, the following class declaration can be used: 

    var Detective = function(name, sidekick){
      this.name = name;
      this.sidekick = sidekick;
      this.notebook = []; 
      this.notebook.push("It was a dark and stormy night.");
    }

A specific Detective object can then be created as follows:

    var detective1 = new Detective("Sherlock Holmes", "Watson");

This calls the constructor, which initializes detective1.name to "Sherlock Holmes", detective1.sidekick to "Watson", and adds one default note to an array that represents this detective's notebook. (Yes, it's the default, because what mystery story doesn't take place on a dark and stormy night?)

So far so good. Once those variables are initialized, though, we're probably going to want to do something with them. That brings us to our next topic: member functions.
