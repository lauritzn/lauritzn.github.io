Classes in JavaScript

As we've already mentioned, classes in JavaScript don't have to be declared with a special class keyword. Instead, a class is defined by the constructor for the objects that belong to the class.

A constructor has the format: 

	  var ClassName = function(value){
		this.member = value;
	    };

The first thing to notice is the use of the this keyword. Just like in C++, the keyword this means that the argument sent to the function value will be assigned to the member property of this object, namely, the object the function is being called on.

The constructor is automatically called whenever a new object of this class is created, on the object that is created. To create the object we use the keyword new.

Here's the general syntax: 

	  var classObject1 = new ClassName("value1");

This creates an object called classObject1 and, by means of the constructor above, assigns a value to classObject1.member.
