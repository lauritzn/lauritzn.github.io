Guide to Emulating Classes in JavaScript

Welcome!

Welcome to this how-to guide for emulating classes in JavaScript! It will show you, step-by-step, how to produce classes in JavaScript that resemble the kind of classes you may already be familiar with from a language like C++. A special technique in JavaScript, called closures, will let us do this.

Why This Guide?

Unlike some other object-oriented programming languages, such as C++, JavaScript does not have a special class keyword for declaring classes. Along with this, it does not have special keywords for public and private data members and functions.

This can make the transition to JavaScript tricky for someone who is used to programming in one of those other languages.
